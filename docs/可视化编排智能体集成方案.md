# 🚀 落地实操步骤（分3个等级）
## 1. 极速原型（1小时）：AgentScope + Gradio
纯 Python，无前端开发，快速实现智能体配置+运行可视化。
```python
# 1. 安装依赖
# pip install agentscope gradio fastapi uvicorn

import agentscope as asc
import gradio as gr

# 2. 初始化 AgentScope 智能体
asc.init(
    model_configs={"config_path": "your_model_config.json"},
    save_dir="./runs"
)
agent = asc.Agent(
    name="Assistant",
    system_prompt="你是一个智能助手",
    model="qwen-max"
)

# 3. Gradio 可视化页面
def chat_with_agent(message):
    return agent(message).content

with gr.Blocks(title="AgentScope 可视化") as demo:
    gr.Markdown("# AgentScope 智能体可视化")
    user_input = gr.Textbox(label="输入")
    output = gr.Textbox(label="回复")
    btn = gr.Button("运行智能体")
    btn.click(chat_with_agent, inputs=user_input, outputs=output)

if __name__ == "__main__":
    demo.launch(server_name="0.0.0.0", server_port=7860)
```

## 2. 最佳可视化（1天）：Flowise + AgentScope
**生产级推荐**，实现真正的拖拽编排：
1. **本地部署 Flowise**
   ```bash
   npm install -g flowise
   flowise start
   ```
2. **封装 AgentScope 为 API**（FastAPI）
   ```python
   from fastapi import FastAPI
   import agentscope as asc
   app = FastAPI()
   asc.init(model_configs={"config_path": "model_config.json"})
   agent = asc.Agent(name="DemoAgent", system_prompt="你是智能助手")

   @app.post("/api/agent/run")
   def run_agent(prompt: str):
       return {"result": agent(prompt).content}
   ```
3. **在 Flowise 中创建自定义节点**，调用上述接口，即可拖拽使用。

# 📌 关键技术注意事项
1. **AgentScope 必须服务化**
   不要用本地脚本，必须封装为 **FastAPI/Flask 接口**，前端才能调用。
2. **通信方案**
   - 配置/保存：HTTP 接口；
   - 运行监控：WebSocket（实时展示智能体对话日志）。
3. **节点设计**
   可视化画布核心节点：
   - 🤖 智能体节点（配置 Agent 参数）
   - 🔧 工具节点（调用 AgentScope 工具）
   - ⚡ 条件节点（工作流分支）
   - 🎬 开始/结束节点
4. **持久化**
   用 SQLite/MySQL 保存编排的工作流配置。
